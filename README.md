## Commands
All commands require OP level 2 permission.
**View Current Status:**
```
/easeon handychest
```
**Enable Handy Chest:**
```
/easeon handychest on
```
**Disable Handy Chest:**
```
/easeon handychest off
```


## Configuration
```json
{
  "enabled": true,
  "requiredOpLevel": 2 // Requires a server restart to take effect.
}
```
